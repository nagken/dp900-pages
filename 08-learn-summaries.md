# DP-900 Microsoft Learn Path Summaries

## Path 1: Explore core data concepts

- Structured / semi-structured / unstructured determines storage choice.
- OLTP for transactions; OLAP for analytics.
- Roles: DBA / engineer / analyst - distinct accountabilities.

## Path 2: Explore relational data services

- Azure SQL DB (PaaS DB) -> single DB cloud apps.
- Azure SQL Managed Instance -> lift-and-shift, cross-DB.
- SQL Server on Azure VM -> full control needed.
- Open-source: Azure Database for PostgreSQL / MySQL.

## Path 3: Explore non-relational data services

- Cosmos DB: 6 APIs (NoSQL, MongoDB, Cassandra, Gremlin, Table, PostgreSQL).
- Five consistency levels - Session is default.
- Storage: Blob (objects), Files (SMB), Queue (msgs), Table (key-value).

## Path 4: Explore data analytics

- Modern data warehouse: ingest -> land -> prep -> serve -> visualize.
- ELT modern; ETL legacy.
- Microsoft Fabric is the all-in-one SaaS choice.
- Power BI: dataset / report / dashboard / paginated report.

## Path 5: Explore real-time data

- Event Hubs / IoT Hub for ingestion.
- Stream Analytics / Fabric Eventstream for processing.
- Real-Time Intelligence (Fabric) / ADX for time-series storage and KQL.
